<hr><h2>HTML Summary</h2> 


<p>This tutorial has taught you how to use HTML to create your own web site.</p> 


<p>HTML is the universal markup language for the Web. HTML lets you format text, add graphics, 


create links, input forms, frames and tables, etc., and save it all in a text file that any 


browser can read and display.</p> 


<p>For more information on HTML, please take a look at our <a href="html_examples.asp"> 


HTML examples</a> and our <a href="/tags/default.asp">HTML reference</a>.</p> 


<hr> 




<h2>Now You Know HTML, What's Next?</h2> 


<hr> 




<h2>Learn CSS</h2> 


<p>CSS lets you style your HTML pages.</p> 


<p>CSS gives you total control of the layout, without messing up the document content.</p> 


<p>To learn more about CSS, please visit our <a href="/css/default.asp">CSS tutorial</a>.</p>

<h2>Learn JavaScript</h2> 


<p>JavaScript makes your website more dynamic. A dynamic website can react to events and allow user interaction.</p> 


<p>JavaScript is the most popular scripting language on the internet and it works with all major browsers.</p> 


<p>If you want to learn more about JavaScript, please visit our <a href="/js/default.asp">JavaScript tutorial</a>.</p>

<p>To make your website available to the world, you must publish it.</p> 


<p>For this, you have two options:</p> 


<ul> 


<li>Use an Internet Service Provider</li> 


<li>Host your own website</li>

<h2>Using an Internet Service Provider</h2> 


<p>An Internet service provider (ISP) is a company that provides services for 


accessing and using the Internet.</p> 


<p>Internet services typically provided by ISPs include Internet access, 


Internet transit, domain name registration, web hosting, Usenet service, and 


colocation.</p> 


<p>Using an Internet Service Provider (ISP) is the most common option. </p> 




<h3>Advantages:</h3> 


<ul> 


<li><strong>Connection Speed</strong> - ISPs have very fast connections to the internet.</li> 


<li><strong>Powerful Hardware</strong> - ISPs have powerful web servers that can be shared by several clients. 


You can also expect an effective load balancing and necessary backup servers 


</li> 


<li><strong>Security and Stability</strong> - ISPs are specialists on web hosting. Expect more than 99% up time, the latest software patches, and the best virus protection</li> 


</ul> 




<h3>Things to Consider:</h3> 




<ul> 


<li><strong>24-hour support</strong> - The ISP should offer 24-hours support. Toll-free phone could also be vital</li> 


<li><strong>Daily Backup</strong> - The ISP must run a daily backup routine</li> 


<li><strong>Traffic Volume</strong> - Check the ISP's traffic volume restrictions (do not end up paying a fortune for unexpected 


high traffic)</li> 


<li><strong>Bandwidth or Content Restrictions</strong> - Check the ISP's bandwidth and content restrictions (Is it possible to publish pictures, video, or sound?)</li> 


<li><strong>E-mail Capabilities</strong> - Make sure the ISP supports the e-mail capabilities you need</li> 


<li><strong>Database Access</strong> - Make sure the ISP supports the database access you need</li> 


</ul> 


<hr> 




<h2>Hosting Your Own Website</h2> 


<p>Hosting your own website, on your own server, is also an option.</p> 




<h3>Things to Consider:</h3> 




<ul> 


<li><strong>Hardware Expenses</strong> - To run a &quot;real&quot; web site, you must buy powerful 


server hardware (a low cost PC will not do the job). You will also 


need a permanent (24/7) high-speed connection</li> 


<li><strong>Software Expenses</strong> - Server-licenses are often higher than client-licenses. 


Server-licenses also might have limits on number of users</li> 


<li><strong>Labor Expenses</strong> - Don't expect low labor expenses. You have to install your own hardware and software. 


You also have to deal with bugs and viruses, and keep your server constantly running</li> 


</ul>
